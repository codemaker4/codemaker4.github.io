Release of the Space Game and the plans for the future.

Midas and I thought that Space Invander was nearing its release. We where thinking about a upgrade system with items and crafting, but we rememberd that the original plan for this game was to be a retro game, simple and fun. To my opinion the Space Game at its current state is a good and fun retro game, so I release it. Of course we will keep improving the game engine and maby make it compatible for mobile devices, but don't expext any other new game changing futures for the game.
I am planning to work on some random smaller projects, catching op with school and maybe start a Youtube channel.

Space Game update 0.2

This update is mainly focused on improving the game engine and how it calculates tickrates.
